Project Overview

Arduino UNO

The brain of our radar system, processing sensor data and controlling components.

HC-SR04 Ultrasonic

Measures distance to objects with high accuracy up to 4 meters.

Servo Motor

Rotates the ultrasonic sensor 180° to scan the environment.

Python Flask

Web interface displaying real-time radar data from Arduino.

System Components

Arduino UNO

Microcontroller board with ATmega328P that processes sensor data and controls all components.

HC-SR04 Ultrasonic

Measures distance by sending ultrasonic pulses and calculating echo return time.

SG90 Servo Motor

Rotates the ultrasonic sensor precisely between 0° to 180° for scanning.

Breadboard

Prototyping board for connecting components without soldering.

LED Indicators

Visual feedback for distance thresholds (green/yellow/red).

Piezo Buzzer

Audible alarm that beeps faster as objects get closer.

How It Works

1. Ultrasonic Pulses

The HC-SR04 emits ultrasonic waves that bounce off objects and return to the sensor.

2. 180° Scanning

The servo motor rotates the sensor to scan the entire area in front of the radar.

3. Real-time Visualization

Flask web server displays the radar data with distance and angle measurements.

Circuit Diagram

Arduino UNO Pins

  • Trig: Digital Pin 9
  • Echo: Digital Pin 10
  • Servo: Digital Pin 11
  • Buzzer: Digital Pin 8
  • LEDs: Digital Pins 2-4

Power Connections

  • 5V to breadboard positive rail
  • GND to breadboard negative rail
  • All components share common ground

Component Specs

  • HC-SR04: 5V, 15mA
  • SG90 Servo: 4.8-6V, 100mA
  • LEDs: 220Ω resistors
  • Buzzer: 5V active type
